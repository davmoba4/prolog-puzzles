
# prolog-puzzles

### Solving the missionary and cannibal crossing problem
 * Install SWI-Prolog
 * Go to the directory where crossing.pl resides through the command line and type:

       $ swipl
       ?- [crossing].
       ?- solve.

* Press ; continuously to go through the valid solutions
* To stop type:

      ?- halt.
