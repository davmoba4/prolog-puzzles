
# prolog-puzzles

### Solving the missionary and cannibal crossing problem
 * Install SWI-Prolog
 * Then go to its appropriate directory through the command line and type:

       $ swipl
       ?- [crossing].
       ?- solve.

* Press ; continuously to go through the valid solutions
* Then type:

      ?- halt.
